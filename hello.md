# hello git

## git 명령어 요약

- clone: 원격 저장소 복사
- add: 스테이지 영역에 작업 파일 추가
- commit: 세이브, 스테이지 영역의 파일들을 가지고 커밋(=세이브) 를 만들 수 있다.
- psuh: 원격 저장소에 커밋을 업로드한다.

## 브랜치 변경

- 브랜치: 기존 내용 유지 새로운 내용 추가하고 싶을 때 사용
- 체크아웃: 특정 브랜치로 돌아가고 싶을 때 사용

## 병합하기 1

- 병합 대상 브랜치가 헤드로부터 시작된 경우
